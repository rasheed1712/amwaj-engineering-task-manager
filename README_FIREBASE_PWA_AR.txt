# Amwaj Engineering Department - Firebase + PWA

## الملفات التي ترفعها على GitHub

ارفع جميع الملفات والمجلدات التالية:
- index.html
- app.js
- manifest.webmanifest
- service-worker.js
- مجلد assets بالكامل

## Firebase

1. Authentication > Sign-in method
   فعل Email/Password.

2. Authentication > Users
   أضف أول حساب يدويًا. أول حساب يدخل إلى البرنامج يصبح Admin.

3. Firestore Database > Rules
   انسخ محتوى firestore.rules.txt واضغط Publish.

## تنزيل البرنامج على الحاسبة

بعد فتح رابط GitHub Pages:
- على Chrome أو Edge سيظهر زر "تنزيل البرنامج" داخل الموقع إذا كان المتصفح يدعم ذلك.
- أو من قائمة المتصفح اختر Install app.
- سيظهر البرنامج كتطبيق على سطح المكتب.

## تنزيل البرنامج على الهاتف

Android Chrome:
- افتح رابط الموقع.
- اضغط قائمة الثلاث نقاط.
- اختر Add to Home screen أو Install app.

iPhone Safari:
- افتح رابط الموقع في Safari.
- اضغط زر المشاركة Share.
- اختر Add to Home Screen.
